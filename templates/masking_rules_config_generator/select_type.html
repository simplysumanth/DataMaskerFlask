Rules Generator

Masking Rules Config Generator enables the user to interact with the GUI and make the desired configurations for files. It includes masking rules and encryption rules for every column in file ,based upon which the user can decide the specifications.



Steps for Masking and Encrypting a CSV file:
  1. Click the config generator button
  2. Provide the path to the CSV file to be masked or encrypted.
  3. Press OK
  4. Make the required configuration in the popup GUI and click OK

    5. Warning: If you want to discard the changes, select the confirm cancel checkbox and press Cancel
Config Generator



Steps for Masking and Encrypting through a Macie Generated Json File:
  1. Click the config generator button
  2. Provide the path to AWS Macie JSON file
  3. Select the CSV file to be masked from the list of CSV files in the dropdown
  4. Press OK
  5. Make the required configuration in the popup GUI and click OK
  6. Warning: If you want to discard the changes, select the confirm cancel checkbox and press Cancel

Macie Config Generator


Steps for Masking and Encrypting directly from AWS S3 Bucket:
  1. Click s3 to generate config button
  2. Provide your AWS Credentials
  3. Select a s3 Bucket from the list of S3 Buckets available in the dropdown
  4. Select the CSV file to be masked from the list of CSV files in the dropdown
  5. Press OK
  6. Make the required configuration in the popup GUI and click OK
  7. Warning: If you want to discard the changes, select the confirm cancel checkbox and press Cancel
